# Assets

This directory contains app assets like images and fonts.

## Required Assets

You should add the following assets for a complete app:
- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image (1284x2778)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48 or larger)

For now, the app will work without these assets but you'll see warnings.
